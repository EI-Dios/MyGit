- Flask框架
- Echarts
- WorldCloud
## 项目简介
> 首先用爬虫爬取豆瓣Top250电影信息，保存在excel和数据库中，并且爬取并下载250部电影海报，将数据库和海报保存至一个新的项目，用Flask框架搭建网站、用Echarts把各电影评分数据图表化、用WorldCloud把各电影词条图像化
